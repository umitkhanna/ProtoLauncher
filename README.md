# ProtoLauncher

Premium marketing site for ProtoLauncher, built with **Next.js 15** (App Router), **Tailwind CSS**, **Framer Motion**, and **Lucide**.

## Scripts

```bash
npm install
npm run dev
npm run build
npm start
```

Set `NEXT_PUBLIC_SITE_URL` in `.env.local` for production canonical URLs, sitemap, and structured data (see `.env.example`).

## Deploy

Optimized for [Vercel](https://vercel.com): connect the repo, add the env var, deploy.
